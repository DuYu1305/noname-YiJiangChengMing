# 关于无名杀

无名杀是一款基于GPLv3协议，使用HTML+JS+CSS制作的一款开源卡牌游戏

开发者：[水乎](https://github.com/libccy)（于2013年底发布）

现更新者：[诗笺](https://github.com/nonameShijian)

本项目对无名杀本体及部分扩展进行魔改然后整合，仅供技术研究与交流使用，请勿作他用，严禁用于任何商业用途，如涉及版权问题，请联系删除。
